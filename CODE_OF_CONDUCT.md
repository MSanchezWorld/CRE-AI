# Code of Conduct

This project follows the Contributor Covenant Code of Conduct.

## Our Standards

Examples of behavior that contributes to a positive environment:

- Being respectful in technical discussions
- Giving and accepting constructive feedback
- Focusing on the project goals and user safety

Examples of unacceptable behavior:

- Harassment or discriminatory language
- Trolling, insults, or personal attacks
- Publishing private information without permission

## Enforcement

Project maintainers are responsible for clarifying and enforcing standards. They may remove, edit, or reject contributions that violate this Code of Conduct.

## Scope

This policy applies within project spaces and public spaces when an individual is representing the project.

## Reporting

Please report violations to the maintainers through the security/contact channel described in `SECURITY.md`.
